# JDBC Student Management System

A simple Java console-based application to manage student records using JDBC and MySQL database connectivity.

## Features
- Add Student
- View Students
- Update Student Details
- Delete Student
- MySQL Database Connectivity using JDBC

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Project Structure
- `Main.java` → Main program
- `Student.java` → Student model class
- `StudentDAO.java` → Database operations
- `DBConnection.java` → Database connection setup
